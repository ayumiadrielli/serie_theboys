Formularios:

Tag label vai se considerada o titulo para input. 

<form>
       <label for="Email">Email</label>

</form>

Dentro de uma Label teremos uma Tag "FOR", nesssa tag vamos passar o ID que queremos atribuir para essa tag. 

    <label for="Email">Email</label>
        <input style="display: block" id="Email" type"email" name="Email" placeholder="coloque um Email, apenas do gmail">

style="display: block" = cria um bloco para inserir o valor
id="Email" = é o valor do for.
name="Email" = nome da label.
Placeholder= e a frase que vai aparecer antes de insirir algo dentro do display:block.
type= email
