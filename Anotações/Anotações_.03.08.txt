Atributos:  
Class: atribui uma classe ao elemento, ou seja, uma class pode ser utilizada um ou mais elementos. 
Id: atribui um elemento unico.
Style: é o elemento que pode incluir o CSS dentro da tag.
Lang: linguagem do codigo. 
Title: titulo.
Alt:um title alternativo, utilizado em imagens. 
hidden: Oculta elementos.
Aling: define padrão de alinhamento.
Width: Define a largura do elemento.
Height: Define altura do elemento.
